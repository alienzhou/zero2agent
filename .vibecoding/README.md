## 目录结构

按 **Stage（阶段）+ Iteration（迭代）** 两层组织：

```
.vibecoding/
├── README.md           # 本文件
├── S01/                # Stage 1: 基础 POC
│   ├── E001/           # 迭代 001: ReACT 基础版
│   │   ├── README.md
│   │   ├── 01-xxx.md
│   │   └── ...
│   ├── E002/           # 迭代 002: ...
│   └── ...
├── S02/                # Stage 2: ...
└── ...
```

**编号规则：**
- `S01` = Stage 01（阶段编号，2 位）
- `E001` = Iteration 001（迭代编号，3 位，每个 Stage 内从 001 开始）
