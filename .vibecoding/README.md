# VibeCoding Records

> 记录与 AI Agent 协作开发的完整过程：提示词、对话、决策、经验。

---

## 目录结构

按 **Epic（阶段）+ Story（迭代）** 两层组织：

```
.vibecoding/
├── README.md           # 本文件
├── E01/                # Epic 1: 基础 POC
│   ├── S001/           # Story 001: ReACT 基础版
│   │   ├── README.md
│   │   ├── 01-xxx.md
│   │   └── ...
│   ├── S002/           # Story 002: ...
│   └── ...
├── E02/                # Epic 2: ...
└── ...
```

**编号规则：**
- `E01` = Epic 01（阶段编号，2 位）
- `S001` = Story 001（迭代编号，3 位，每个 Epic 内从 001 开始）
