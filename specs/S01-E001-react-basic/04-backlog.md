## 技术优化 Backlog

### 安全性

| 优化项 | 说明 | 原因 |
|--------|------|------|
| 路径安全检查 | 禁止访问项目外文件 | MVP 阶段暂不做 |
| 路径穿越防护 | 检测 `../` 等路径 | MVP 阶段暂不做 |
| 敏感文件过滤 | 不读取 `.env` 等 | MVP 阶段暂不做 |

### 性能

| 优化项 | 说明 | 原因 |
|--------|------|------|
| Token 计数 | 统计消耗的 token | 实现复杂 |
| 大文件分片 | 自动分片读取大文件 | 实现复杂 |
| 缓存机制 | 缓存已读取的文件 | 过早优化 |

### 架构

| 优化项 | 说明 | 原因 |
|--------|------|------|
| Tool Registry | 工具注册与发现机制 | MVP 阶段不需要抽象 |
| Plugin 系统 | 支持动态加载工具 | MVP 阶段不需要 |
| 多 LLM 支持 | 统一 Adapter 抽象 | 先跑通一个 SDK |

## 被拒绝的方案
